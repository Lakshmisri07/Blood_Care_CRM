## Overview
BloodCare CRM is a Salesforce-based application developed as part of an internship project. It helps manage blood banks, donors, and blood requests efficiently using Salesforce declarative features.

## Key Features
- Custom Objects: Blood Bank, Donor, Blood Request
- Lookup Relationships
- Validation Rules
- Record-Triggered Flow for emergency requests
- Reports and Dashboards
- Custom Lightning App
